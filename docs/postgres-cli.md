### Postgres CLI 

##### List all users and its role
```sql
\du
```

##### List all functions
```sql
\df
```

##### List schemas 
```sql 
\dn
```

##### List indexes 
```sql
\di
```

##### List tables 
###### From all schemas 
```sql 
\dt *.*
```

###### From a schema 
```sql 
\dt schema.*
```

##### List all views 
```sql
\dT+
```

##### List all views 
```sql
\dv
```
